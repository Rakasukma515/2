# Event Ticketing System

Sebuah aplikasi pemesanan dan pembelian tiket event secara online berbasis mobile. Database yang digunakan Firebase. Versi Flutter 3.3.8.

## Roles

* Peserta

## Fitur

* Fitur Autentikasi
* Fitur Pencarian
* Fitur Pemesanan
* Fitur Pembelian
* Fitur History
* Fitur QR Code
